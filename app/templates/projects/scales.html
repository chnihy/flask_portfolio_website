Scales

{% endblock project_title %}

Scales.py (also Scales.js) is a simple program for building scales

>>> python3
>>> import scales
>>> cmajor = scales.Major('c')
>>> cmajor.scale
['C', 'D', 'E', 'F', 'G', 'A', 'B']

Currently supported scales:

  • Chromatic
  • Major
  • Minor
  • MelodicMinor
  • MajorPentatonic
  • MinorPentatonic

Check out the GitHub for a more detailed explanation
