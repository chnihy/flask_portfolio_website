Notereader is a sight reading exercise generator built with Python and Kivy

See on GitHub

Built using Python and Kivy UI as well as Scamp for engraving. Notereader generates XML files which can be read by a music notation software like Sibelius or Musescore. Or you can open directly in your OS using LilyPond/Abjad engraving.

There are a lot of controls over the output, including:

  • Time Signature/Tempo
  • Rhythmic Subdivision
  • Scale/Clef/Key
  • Sample size - aka the number of notes from the scale
  • The number of exercises

This app has great applications for music lessons - teachers or students can use this daily to increase their sight reading skills.
